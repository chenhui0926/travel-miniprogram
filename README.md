# 🎒 特种兵旅游助手 - 微信小程序

一个专为"特种兵式旅游"设计的微信小程序，支持行程规划、地图导航、天气感知和进度追踪。

> 📖 **新手配置指南**：打开项目里的 `setup-guide.html`（用浏览器打开），跟着步骤一步步操作即可。

## 功能特性

- **智能行程规划**：选择目的地、天数、风格偏好，自动生成最优路线
- **自定义目的地**：支持任意城市，手动添加打卡点
- **实时天气感知**：接入和风天气 API，雨天自动调整室外景点安排
- **地图导航**：接入腾讯地图，支持路线规划和一键导航
- **进度追踪**：打卡记录、完成度统计、下一步智能推荐
- **行程分享**：一键分享给同行人

## 快速开始

### 1. 申请 API Key

#### 和风天气 Key（天气数据）
1. 访问 https://dev.qweather.com/
2. 注册账号并创建应用
3. 获取 API Key
4. 免费版支持：实时天气 + 3天预报

#### 腾讯位置服务 Key（地图导航）
1. 访问 https://lbs.qq.com/dev/console/application/mine
2. 注册并创建应用（选择"微信小程序"类型）
3. 获取 Key
4. 在小程序后台添加 request 合法域名：`https://apis.map.qq.com`

### 2. 配置 Key

修改 `app.js` 中的配置项：

```javascript
App({
  globalData: {
    TENCENT_MAP_KEY: '你的腾讯地图Key',
    QWEATHER_KEY: '你的和风天气Key'
  }
});
```

### 3. 导入微信开发者工具

1. 下载安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 选择"导入项目"
3. 选择 `travel-miniprogram` 目录
4. 填写你的 AppID（或选择测试号）
5. 在开发者工具中点击"详情"→"本地设置"→勾选"不校验合法域名"（开发时）

### 4. 发布前配置

在微信公众平台小程序后台：
- 添加服务器域名：`https://devapi.qweather.com` 和 `https://geoapi.qweather.com`
- 添加服务器域名：`https://apis.map.qq.com`
- 添加位置权限说明

## 技术说明

### 雨天自动调整逻辑
当检测到雨天时，`cityData.js` 中的 `adjustForWeather` 函数会：
1. 遍历每天的行程
2. 识别室外类型景点（`景点`、`自然`）
3. 在同一天内寻找室内景点进行时间互换
4. 在调整后的景点上标注天气调整说明

### 地图导航流程
1. 用户点击"导航"按钮
2. 获取用户当前位置（gcj02坐标系）
3. 调用腾讯路线规划 API 计算距离和时间
4. 展示路线信息，用户确认后唤起地图导航

### 数据持久化
- 行程状态保存在 `wx.setStorageSync('travel_state')`
- 重启小程序后自动恢复上次行程
- 首页提供"继续行程"或"新建行程"选项
