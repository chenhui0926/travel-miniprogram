API Key 申请 & 微信开发者工具配置指南

1
申请和风天气 API Key
用于获取目的地实时天气和 3 天预报
1.1 注册账号
  1. 打开浏览器，访问 dev.qweather.com
  2. 点击右上角「注册」按钮
  3. 使用邮箱注册，验证邮箱后登录
1.2 创建应用获取 Key
  1. 登录后进入「控制台」
  2. 左侧菜单点击「应用管理」→「创建应用」
  3. 应用名称填：特种兵旅游助手
  4. 选择 Key 类型：Web API
  5. 创建成功后，复制你的 Key（一串字母数字）
免费版额度：1000 次/天，个人使用完全够用。
1.3 保存你的 Key

把复制到的 Key 先保存在记事本里，等下会用到。

你的和风天气 Key 大概长这样：
abc123def456ghi789jkl012mno345pq
2
申请腾讯位置服务 Key
用于地图展示、路线规划、一键导航
2.1 注册/登录
  1. 打开浏览器，访问 lbs.qq.com
  2. 用 QQ 或微信扫码登录
  3. 首次使用需要完成实名认证（按页面提示操作即可）
2.2 创建应用
  1. 进入控制台后，点击「应用管理」→「创建应用」
  2. 应用名称填：特种兵旅游助手
  3. 应用类型选择：微信小程序
  4. 填写你的小程序 AppID（如果还没有，先选"测试号"，后面可以改）
2.3 启用所需 API
  1. 在应用详情页，点击「添加 Key」
  2. 勾选需要启用的 WebService API：
    • ✅ 路线规划（步行/驾车/公交）
    • ✅ 地点搜索
    • ✅ 逆地址解析（坐标转地址）
  3. 点击「提交」，复制生成的 Key
免费版额度：10000 次/天，路线规划完全够用。
2.4 配置小程序后台域名
  1. 登录 微信公众平台
  2. 进入你的小程序后台 →「开发」→「开发管理」→「开发设置」
  3. 找到「服务器域名」→「request 合法域名」
  4. 添加以下两个域名：
    https://apis.map.qq.com
    https://apis.map.qq.com/ws/direction/v1/
  5. 保存后等待微信审核（通常几分钟）
如果不配置域名，真机调试时地图 API 会请求失败。开发阶段可以在开发者工具里勾选「不校验合法域名」临时绕过。
3
填入 API Key 到项目
把申请到的两个 Key 替换进代码
3.1 打开 app.js

用任意代码编辑器（VS Code、记事本等）打开项目根目录的 app.js 文件。

3.2 替换两个 Key

找到这两行，把引号里的内容替换成你刚才申请的真实 Key：

TENCENT_MAP_KEY: '请替换为你的腾讯地图Key',
QWEATHER_KEY: '请替换为你的和风天气Key',

替换后大概长这样：

TENCENT_MAP_KEY: 'ABCD-EFGH-IJKL-MNOP',
QWEATHER_KEY: 'abc123def456ghi789',
3.3 保存文件

按 Ctrl+S（Windows）或 Cmd+S（Mac）保存。

4
导入微信开发者工具
编译预览你的小程序
4.1 下载安装开发者工具
  1. 访问 微信开发者工具下载页
  2. 根据你的系统（Windows/Mac）下载对应版本
  3. 安装并打开
4.2 导入项目
  1. 打开微信开发者工具，点击「+」或「导入项目」
  2. 点击「目录」旁边的选择按钮，找到并选中 travel-miniprogram 文件夹
  3. AppID 选项：
    • 如果你有小程序 AppID，直接填入
    • 如果没有，选择「测试号」（功能受限但够用）
  4. 项目名称填：特种兵旅游助手
  5. 点击「确定」导入
4.3 开发配置（重要）
  1. 导入成功后，点击顶部菜单「详情」按钮（或按 Ctrl/⌘ + Shift + P）
  2. 在「本地设置」标签页，勾选以下两项：
    • ☑️ 不校验合法域名、web-view...
    • ☑️ 启用代码自动热重载
  3. 关闭详情面板，等待编译完成
「不校验合法域名」只在开发阶段勾选。正式发布前务必在微信小程序后台配置好合法域名，并取消勾选此项。
4.4 编译运行
  1. 点击工具栏的「编译」按钮（或按 Ctrl/⌘ + B）
  2. 左侧模拟器应该能看到首页界面
  3. 点击「预览」可以生成二维码，用手机微信扫码真机测试
第一次编译可能需要 10-30 秒，请耐心等待。

?
常见问题排查
地图导航点击没反应？
  • 检查 app.js 中的 TENCENT_MAP_KEY 是否已替换
  • 检查是否勾选了「不校验合法域名」
  • 真机预览需要在小程序后台配置 request 合法域名
天气显示不出来？
  • 检查 app.js 中的 QWEATHER_KEY 是否已替换
  • 和风天气 Key 需要几分钟才能生效，申请后稍等一会儿再试
  • 检查城市名称是否正确（尽量用中文全称）
编译报错？
  • 确认项目路径没有中文或特殊字符
  • 确认微信开发者工具是最新版本
  • 尝试点击「工具」→「构建 npm」（如果有用到 npm 包）
真机预览位置获取失败？
  • 在真机上第一次会弹窗请求位置权限，请点击「允许」
  • 检查手机定位服务是否开启
  • 微信需要获取位置权限，在「设置」→「隐私」→「位置」中检查
